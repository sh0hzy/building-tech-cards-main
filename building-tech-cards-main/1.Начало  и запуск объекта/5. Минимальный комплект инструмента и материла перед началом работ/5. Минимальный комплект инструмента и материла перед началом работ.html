Универсальные предметы:
  • Ножовка по дереву.
  • Разметочный шнур.
  • Рулетка 3 шт.
  • Перманентные маркеры.
  • Карандаши строительные.
  • Канцелярский нож и набор лезвий.
  • Скарпель для перфоратора 2 шт.
  • Лопатка для перфоратора
  • Отвес
  • Уровень лазерный
  • Уровень строительный
  • Разводные ключи.
  • Диски для болгарки 10-15 шт.
  • Пистолет для герметика.
  • Доски, брусья. Доски (укосины и ребра жесткости), не строганные, толщина 20 мм., ширина 100-150 мм. Брусья (каркас), не строганные, сечение 50х50 мм.
  • Стремянка (5-7 ступеней) 2 шт. Стремянка используется до 3-й ступени, чтобы при упоре ног оставалась площадка для необходимого материала. Подбирается под заданную высоту.
  • Легкий строительный козел.
  • Миксер 2 шт (+ 1 дешевый запасной). Покупать усиленный. При покупке обращать внимание на хвостовик, может понадобиться переходник.
  • Ведра 60 литров 3-5 шт. Для штукатурки и стяжки.
  • Ведра 20 литров 5-10 шт. Для шпаклевки, мелкие замесы.
Защита поверхностей:
  • Оргалит. Защита мест общего пользования.
  • Пеноплекс 20 мм. Защита входной двери и мест общего пользования.
  • Ветошь. Коврик при входе и выходе, чтобы не пачкать пол. Также защита радиатора отопления и напольных чистовых покрытий.
  • Полиэтиленовая техническая плёнка (лучше 100 микрон) 2-3 рулона
  • Линолеум. Защита пола в местах общего пользования.
  • Пузырчатая пленка - защита радиатора отопления
Предметы для уборки:
  • Строительный пылесос (7-8 мешков на смену)
  • Совковая лопата
  • Уборка крупного мусора.
  • Веник
  • Совок
  • Грубая щётка 2 шт.
  • Средняя грубость (для потолка), высокая грубость.
  • Садовый опрыскиватель 2 шт.
  • Прибивать пыль и наносить грунт.
  • Мешки для мусора 100 шт. Прочные мешки для мусора (белые) в большом количестве. Зеленые мешки для мусора в меньшем количестве (не прочные) - для не острого мусора.
  • Бак 90 литров.
  • Для жидких отходов. Важно не накапливать больше 50% заполняемости - потом невозможно будет сдвинуть и удалить мусор.
Временная электрика:
  • Кабель 1,5 мм2
  • Кабель 2,5 мм2
  • Лампочки E27 60 ватт белый лед энергосберегающая
  • Лампа светодиодная REV E27 6500К 50 Вт 4000 Лм 180-240 В цилиндр T125 прозрачная.
  • Патроны E27
  • Удлинители
  • Колодки с выключателем.
  • Выключатели
  • Автоматы
  • Хомуты / стяжки / клеммы
  • Инструменты для монтажа временной электрики
  • WAGO клемники тройные.
  • Вилки прямые каучук.
  • Изолента 2 цвета.
  • Распаечные коробки 10х10.
  • Включатели одноклавишные накладные.
Временная сантехника:
  • Временный унитаз с бачком (гофра, гибкая подводка для смыва)
  • Временная мойка (пластиковый бак для ее изготовления).
  • Лён, паста, сифон для подключения.
  • Временный поддон или таз 90 литров
  • Смеситель
  • Душевая лейка
  • Сантехнические резиновые прокладки 10-15 шт.
  • Пескоуловитель на 25 литров.
  • Водопроводные трубы.
  • Силиконовый герметик.
  • Поддон душевой.
  • Сифон для поддона
  • Фум-лента.
  • Бруски для изготовления каркаса временной мойки и дешевой.
Инвентарь для кладки из блоков:
  • Строительный угольник
  • Слесарный угольник
  • Кельма
  • Зубчатый шпатель
  • Обычный шпатель
  • Усиленный шпатель
  • Резиновый молоток
  • Ножовка для блоков/полотно для сабельной пилы. При работе не давить на полотно и не перегревать.
Инвентарь для малярных работ:
  • Колер. Для более красивого результата стоит окрашивать потолок перед монтажом электрики.
  • Шапочки для душа
  • Ванночки для валиков
  • Набор валиков
  • Кисти разной ширины
  • Телескопическая ручка для валика 2 шт.
  • Аэрозольная краска 1-3 шт. Покраска металлических деталей, стояки, трубы, предварительная разметка будущих стен.
  • Монтажная пена профессиональная для пистолета
  • Пистолет для монтажной пены
  • Очиститель монтажной пены
Крепёж:
  • Саморезы универсальные и по дереву 25, 35, 45, 55 мм (для скручивания брусьев 71, 92 мм)
  • Дюбель-гвозди 6 мм (длина 35 мм/45 мм)
  • Подвесы
  • Универсальный крепеж.
  • Скотч (малярный, армированный, обычный, деликатный). Не должен заканчиваться. Нужно очень большое количество.
Средства защиты:
  • Перчатки - обычные прорезиненные. Много.
  • Защитные очки - несколько пар.
  • Респираторы, не пропускающие мелкодисперсную пыль (лучше со съемными фильтрами).
  • Строительные наушники
Электроинструмент
  • Перфоратор малый, большой.
  • Дрель с шуруповертом.
  • Миксер строительный.
  • Строительный пылесос с циклоном.
  • УШМ болгарка.
  • Штроборез.
  • Сабельная пила.
  • Строительный фен.
  • Отбойный молоток.
  • Дрель с насадкой для шлифовки.
  • Лобзик.
